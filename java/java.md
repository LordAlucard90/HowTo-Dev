## Installation

#### Linux

```bash
# java runtime rnvironment
sudo apt install openjdk-8-jre  # java  8
sudo apt install openjdk-11-jre # java 11

# java development kit
sudo apt install openjdk-8-jdk  # java  8
sudo apt install openjdk-11-jdk # java 11
```
